API Documentation

Complete guide to all available endpoints

Base URL

https://mongotest2026.vercel.app

Available Endpoints

GET /api/foods

Retrieve all Nigerian foods in the database

Response Example

{
  "success": true,
  "status": 201,
  "count": 30,
  "data": [
    {
      "id": 1,
      "name": "Jollof Rice",
      "description": "A popular West African dish...",
      "category": "Main Course",
      "region": "All regions",
      "ingredients": ["Rice", "Tomatoes", ...],
      "preparationTime": "45-60 minutes",
      "difficulty": "Medium",
      "isVegetarian": true,
      "isSpicy": true,
      "servingSize": "4-6 people",
      "calories": 350,
      "price": 1500

}
GET /api/foods/:id

Get a specific food by its ID

Parameters

Parameter Type Description
id Integer The ID of the food (1-30)

Example Request

GET /api/foods/1
GET /api/foods/category/:category

Filter foods by category

Available Categories

Main Course Soup Swallow Snack/Street Food Breakfast/Snack Side Dish Appetizer/Side Dish Snack/Dessert

Example Request

GET /api/foods/category/soup
GET /api/foods/region/:region

Filter foods by regional origin

Available Regions

All regions Northern Nigeria South-West Nigeria South-South Nigeria South-East Nigeria

Example Request

GET /api/foods/region/northern
GET /api/foods/filter/vegetarian

Get all vegetarian-friendly foods

Example Request

GET /api/foods/filter/vegetarian
GET /api/foods/filter/spicy

Get all spicy foods

Example Request

GET /api/foods/filter/spicy

Code Examples

Integration examples in popular languages

Fetch API (JavaScript)

// Get all foods
fetch('https://mongotest2026.vercel.app/api/foods')
  .then(response => response.json())
  .then(data => {
    console.log(data);
    console.log(`Total foods: ${data.count}`);
  })
  .catch(error => console.error('Error:', error));

// Get food by ID
fetch('https://mongotest2026.vercel.app/api/foods/1')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

// Using async/await
async function getFoods() {
  try {
    const response = await fetch('https://mongotest2026.vercel.app/api/foods');
    const data = await response.json();
    return data;
  } catch (error) {
    console.error('Error:', error);
  }
}

Python (requests library)

import requests

# Get all foods
response = requests.get('https://mongotest2026.vercel.app/api/foods')
data = response.json()
print(f"Total foods: {data['count']}")

# Get food by ID
response = requests.get('https://mongotest2026.vercel.app/api/foods/1')
food = response.json()
print(f"Food name: {food['data']['name']}")

# Get foods by category
response = requests.get('https://mongotest2026.vercel.app/api/foods/category/soup')
soups = response.json()
for soup in soups['data']:
    print(soup['name'])

PHP (cURL)

<?php
// Get all foods
$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, 'https://mongotest2026.vercel.app/api/foods');
curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);
$response = curl_exec($ch);
curl_close($ch);

$data = json_decode($response, true);
echo "Total foods: " . $data['count'];

// Get food by ID
$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, 'https://mongotest2026.vercel.app/api/foods/1');
curl_setopt($ch, CURLOPT_RETURNTRANSFER, 1);
$response = curl_exec($ch);
curl_close($ch);

$food = json_decode($response, true);
echo $food['data']['name'];
?>
